# rontetest

## Requirements

- PHP >= 7.0


## Running

Simplest way

```
php ./rshort.php -inputFile=./hotels.csv -outputTo=JSON=json.json
```

All options available in short help

```
php ./rsort.php -help
```

All option actions coded by ControlParametersProcessorClass


## Running tests

```
./phpunit tests/AllTests.php
```
